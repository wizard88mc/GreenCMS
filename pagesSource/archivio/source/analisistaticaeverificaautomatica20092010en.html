Contents

The main topics of the course are as follows:

Introduction to static analysis.
Approximate semantics.
Semantic approximation by abstract interpretation.
Abstract domains and abstract fixed points computations.
Abstract interpretation of programs.

Data flow analysis.
Live variables, reaching definitions, available expressions, very busy expressions, copy propagation.
General framework of dataflow analysis.
Distributive vs non-distributive dataflow analysis.

Introduction to automatic verification.
Models of reactive systems.
Specification languages for temporal properties of system models: CTL, LTL.
Verification by model checking.
CTL and LTL verification algorithms.
Abstract model checking.
Model reduction by behavioural equivalences (bisimulation, simulation).
