Contents

The course illustrates the basic concepts of databases and the techniques for their design and implementation, with special interest in relational databases. The covered topics are: Databases and database management systems; Conceptual design of a database in the object model; logical design of a database in the relational model; Relational model, relational algebra and domain relational calculus; Normalisation; The SQL language: DDL and DML; Using the MySQL DBMS. SQL for applications: embedded SQL and call level interface (ODBC, JDBC); Web access to databases: PHP and dynamic web pages.
