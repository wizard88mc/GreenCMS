Contents

I punti principali del corso sono i seguenti:
-il linguaggio funzionale ML
-inferenza dei tipi
-polimorfismo
-ruolo dei tipi nei linguaggi di programmazione
-gestione run time di blocchi e funzioni
-le funzioni come valori qualsiasi
-approfondimenti su C++ e Java
-progetto in cui si realizza un compilatore e un interprete di un semplice linguaggio funzionale
The major points of the course are as follows:
-the functional language ML
-type inference procedure
-polymorphism
-role of types in programming languages
-run time managment of blocks and functions
-C++ and Java in depth
-a project that consists in building compiler and interpreter for a simple functional language
