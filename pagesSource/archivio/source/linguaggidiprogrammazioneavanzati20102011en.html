Contents

this course illustrates some of the advanced techniques of modern programming languages, such that multi-paradigm programming languages, interoperability between languages, design-by-contracts, advanced type system techniques, high level programming support for concurrent, distributed and web-based programming. These concepts will be studied in the context of the following languages: C#, Spec#, Python, Scala, Ruby, Erlang, Go, Ruby on Rails.
