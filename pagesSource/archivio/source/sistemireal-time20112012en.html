Contents

Class lectures examine the software architecture of real-time embedded systems, paying particular attention to their design, implementation and verification. The issues that are discussed include:
- architectural characterization (at hardware, software and system level)
- time management and handling of hardware interfaces
- paradigms for the design and programming of real-time embedded software
- techniques and approaches for model-based analysis of real-time systems
- verification and validation.
As part of the class, students will be confronted with practicals that facilitate the familiarization with some of the some of the key issues in the domain.
