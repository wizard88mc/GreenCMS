Contents

1. Introduction.
The concept of hypertext, the World Wide Web and Internet. The W3C consortium, Client-Server architectures and Internet protocols.

2. The languages of the static web.
XHTML language and the style sheets (CSS): formatting of text and graphics on Web; links and navigation.

3. The XML.
EXtensible Markup Language (XML), languages for the definition of a schema (DTD and XMLSchema), hints at retrieving data (XPath) and introduction to the processing style sheets for XML (XSLT).

4. The languages for the dynamic web (Programming on the Internet).
Javascript Client-Side and Server-Side. Common Gateway Interface (CGI). The Perl language.
