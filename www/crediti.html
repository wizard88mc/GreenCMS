Sviluppo

Questo sito è stato sviluppato come progetto di stage interno del Corso di Laurea in Informatica. Le tecnologie utilizzate sono:

  • XHTML Strict 1.0
  • CSS 2.1
  • CSS 3 (alcune regole)
  • XML
  • XSLT
  • Perl
  • Javascript e Ajax

Si sono seguite le direttive WCAG 2.0 AAA per cui il sito risulta accessibile e navigabile da qualsiasi supporto.

Tutte le tecnologie web elencate sono state studiate ed approfondite dagli studenti durante il corso di Tecnologie Web del corso di Laurea in Informatica
